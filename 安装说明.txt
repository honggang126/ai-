# 写小说软件安装说明（交流群： QQ群：1035396790）


## 一、环境要求
- **Python版本**：Python 3.6 及以上（推荐Python 3.8+，兼容性更好）
- **操作系统**：Windows、macOS、Linux 均可运行（需对应系统的Python版本）


## 二、安装步骤

### 1. 安装Python
- 访问Python官网：https://www.python.org/downloads/，下载对应系统的Python 3.6+版本
- 安装时注意勾选 **"Add Python to PATH"**（Windows系统），macOS/Linux通常会自动配置环境变量
- 验证安装：打开命令行（Windows：CMD/PowerShell；macOS/Linux：终端），输入以下命令，显示版本号即安装成功
  ```bash
  python --version  # 或 python3 --version（部分系统）
  ```


### 2. 安装依赖库
该软件依赖以下第三方库，需通过pip安装：
- PyQt5：用于图形界面开发
- requests：用于网络请求（可选，代码中主要用于激活验证）

打开命令行，执行以下命令：
```bash
# 安装PyQt5（核心依赖）
pip install PyQt5 -i https://pypi.douban.com/simple/

# 安装requests（网络请求依赖）
pip install requests -i https://pypi.douban.com/simple/
```

> 说明：`-i https://pypi.douban.com/simple/` 是使用国内豆瓣镜像源，加快下载速度，如遇安装失败可尝试去掉该参数或更换其他镜像源（如阿里云、清华源等）


### 3. 准备软件文件
- 将 `写小说软件_4.py` 保存到本地任意目录（建议路径不含中文和特殊字符，如 `D:\novel_software\`）
- （可选）若需要窗口图标，可在软件同目录下放置名为 `1.png` 的图片文件（不放置也不影响运行，仅图标无法显示）


## 三、运行软件
1. 打开命令行，切换到软件所在目录：
   ```bash
   # 示例（Windows）：
   cd D:\novel_software
   
   # 示例（macOS/Linux）：
   cd /Users/yourname/novel_software
   ```

2. 执行命令运行软件：
   ```bash
   python 写小说软件_4.py  # 或 python3 写小说软件_4.py（部分系统）
   ```

3. 首次运行会显示激活界面，输入有效的激活码即可使用（代码中内置测试激活码 `demo1234567890` 可用于测试）


## 四、常见问题解决
1. **PyQt5安装失败**：
   - 确保Python版本符合要求
   - 尝试更新pip：`pip install --upgrade pip`
   - 对于Linux系统，可能需要额外安装系统依赖：`sudo apt-get install python3-pyqt5`（Ubuntu/Debian）

2. **运行时提示缺少模块**：
   - 检查是否漏装依赖，重新执行步骤2的安装命令

3. **网络验证失败**：
   - 确保网络连接正常，防火墙未阻止软件访问网络
   - 若无法访问外部网络，可使用测试激活码 `demo1234567890`

4. **中文显示乱码**：
   - 确保系统已安装中文字体
   - 尝试更换Python版本（推荐3.8+版本对中文支持更好）


## 五、注意事项
- 软件激活信息保存在用户目录下的 `.your_app_license` 文件和同目录的 `activation_code.txt` 中
- 如需卸载，直接删除软件文件及上述两个配置文件即可
- 请勿修改代码中的激活验证逻辑，否则可能导致软件无法正常使用
